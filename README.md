# JSON-VIEWER（JSON可视化工具）



## 技术栈：Vue 3 + Vite +TS + [Antv/G6](https://g6.antv.antgroup.com/zh) + Pinia +[vue3-json-editor](https://www.npmjs.com/package/vue3-json-editor)



## 实现功能：

1. JSON编辑生成对应视图
2. 布局配置调整更新视图
3. 节点收起、展开
4. 导入、导出JSON
5. 指定额外解析字段
6. 4种主题随意切换
7. 视图节点聚焦搜索
8. 视图导出为图片
9. 点击节点查看节点详情
10. 画布操作（放大/缩小/居中/全屏/退出全屏）
